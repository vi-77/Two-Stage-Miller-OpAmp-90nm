Project Goal: Design a high-gain, stable 2-stage Op-Amp for 90nm CMOS technology.
Technical Specifications:DC Gain: > 60dB
Unity Gain Bandwidth (UGB):20 MHz
Phase Margin:> 60° (Stable)
Load Capacitance: 5pF
Key Design Features:Architecture: NMOS differential pair input with a PMOS active load.
Compensation: Miller compensation using a Cc capacitor.
Stability Optimization: Integrated a Nulling Resistor to cancel the Right Half Plane (RHP) zero.
Tools: Designed and simulated using Cadence Virtuoso (Spectre).
